# Task-01: Text Generation using GPT-2

## 📌 Overview
This project demonstrates text generation using GPT-2, a transformer-based language model developed by OpenAI.

## 🧠 Concept
GPT-2 is a deep learning model trained on large text data that can generate human-like text based on input prompts.

## ⚙️ How it works
- Load pre-trained GPT-2 model
- Feed custom dataset (data.txt)
- Generate contextual text using probabilistic prediction
- Save generated output

## 📂 Files
- gpt2_text_generation.ipynb
- data.txt
- generated_output.txt
- requirements.txt

## 📦 Libraries
- Transformers
- PyTorch
- Datasets

## 🎯 Output
The model generates coherent and context-aware text based on training data.
